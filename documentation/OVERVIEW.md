# IronSnake

# Roles
- Teachers
- Students
- Administrators

# Main features
## Teachers need to be able to: 
- Write tasks
- See students' submissions
- Correct students' submissions manually OR automatically 
- Provide feedbacks to students
- Upload new exercices using an online editor, WebDAV, or Git

## Students need to be able to: 
- Submit their work
- See their teacher's feedback

## Administrators need to be able to: 
- Create courses in bulk
- Manage users in bulk
- Configure the mapping of submissions towards execution pods
