# CS01 - Introduction to Computer Science

## Course Description

This course provides an introduction to computer science and programming concepts. Students will learn the basics of programming languages, algorithms, data structures, and software development methodologies. The course emphasizes hands-on experience through programming assignments and projects.

## Course Objectives

By the end of this course, students will be able to:

- Understand the fundamental concepts of computer science and programming.
- Write simple programs using a programming language.
- Analyze and design algorithms.
- Implement data structures.
- Collaborate effectively in software development teams.

## Course Materials

- Lecture slides and videos
- Programming assignments and projects
- Textbook: "Introduction to Computer Science" by John Doe

## Course Schedule

- Week 1: Introduction to Computer Science and Programming
  - Introduction to Python syntax and basic concepts
  - Variables, data types, and operators
  - Task 01: Binary to Base64 conversion
  - Task 02: Base conversions quiz
- Week 2: Algorithms and Complexity
  - Algorithm design and analysis
  - Big O notation and complexity analysis
  - Sorting algorithms (Bubble Sort, Quick Sort)
  - Task 04: Bubble Sort implementation
- Week 3: Data Structures
  - Linear structures: Arrays, Linked Lists, Stacks, Queues
  - Non-linear structures: Trees, Hash Tables
  - Task 05: Data structures quiz
- Week 4: Recursion and Advanced Algorithms
  - Recursive thinking and base cases
  - Fibonacci, factorial, and power functions
  - Divide and conquer strategies
  - Task 07: Recursive algorithms implementation
- Week 5: Object-Oriented Programming
  - Classes, objects, and methods
  - Encapsulation, inheritance, polymorphism, abstraction
  - Design patterns introduction
  - Task 08: OOP concepts quiz
- Week 6: Text Processing and Analysis
  - String manipulation and regular expressions
  - File I/O operations
  - Data parsing (CSV, JSON)
  - Task 06: Word frequency analysis
- Week 7-8: Final Project
  - Team formation and project selection
  - Implementation and testing
  - Code review and refactoring
  - Final presentation and demonstration

## Course Evaluation

- Homework assignments (30%)
- Midterm exam (30%)
- Final project (40%)
